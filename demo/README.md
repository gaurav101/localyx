# Localyx Demo

This demo showcases:

- Basic persistent state + same-tab/cross-tab sync.
- TTL sessions with `absolute` and `sliding` strategies.
- `onExpire` callback behavior.
- Remaining TTL display using `getRemainingTtl(...)`.
- Custom encrypt/decrypt usage.

## Run

```bash
npm install
npm run dev
```

Open the printed Vite URL.
